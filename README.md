# Marketing Campaign ROI Analyzer with Salesforce Integration

## 📌 Project Overview

This project analyzes marketing campaign performance using Power BI with a Salesforce-inspired dataset. The dashboard helps marketing teams evaluate campaign effectiveness by tracking spending, lead generation, conversions, revenue, and Return on Investment (ROI).

The project demonstrates how business intelligence can support data-driven marketing decisions through interactive visualizations and KPI monitoring.

---

## 🎯 Objectives

- Analyze campaign performance across different marketing channels.
- Measure campaign ROI and conversion effectiveness.
- Track marketing spend and generated revenue.
- Compare campaign performance across regions.
- Provide actionable insights for marketing decision-making.
